# poong

## Comment l'utiliser ?

Premièrement, [installez Rust](https://rust-lang.org/tools/install/). Ensuite, clonez le repo Git :

```
git clone https://github.com/science-numerale/poong.git
```

Déplacez-vous dans le dossier racine, puis lancez :

```
cargo run
```

... et découvrez l'incroyable univers Poong !
