# poong

## Comment l'utiliser ?

La méthode la plus simple est de télécharger le fichier binaire compilé correspondant à votre système d'exploitation *via* l'espace versions de GitHub (voir à droite).

Ou alors, pour compiler manuellement, [installez Rust](https://rust-lang.org/tools/install/), puis clonez le repo Git avec :

```sh
git clone https://github.com/science-numerale/poong.git
```

Déplacez-vous dans le dossier racine, puis lancez :

```sh
cargo run
```

... et découvrez l'incroyable univers Poong ! (!!!)
